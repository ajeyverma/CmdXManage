PowerShell Commands List

Get-ChildItem -Force @args

Lists all files and folders including hidden ones.

Set-Location <path>

Changes current directory to <path>.

Set-Location $env:USERPROFILE<path>

Changes directory to a folder inside user profile.

Set-Location @args

Change your directory (like cd).

Clear-Host

Clears the console screen.

New-Item -ItemType Directory -Path <folderName>

Creates a new directory.

New-Item -ItemType Directory $args

Creates a new directory (like mkdir).

Remove-Item @args

Deletes files or folders.

Get-Content @args

Shows the content of a file.

& 'full_path_to_executable' @args

Runs an external program or executable (e.g., launching VS Code).

Copy-Item @args

Copies files or folders.

Move-Item @args

Moves or renames files or folders.

Rename-Item -Path <old> -NewName <new>

Renames a file or folder.

Test-Path <path>

Checks if a file or folder exists.

Get-Process

Lists running processes.

Stop-Process -Id <processId>

Stops a running process by its ID.

Get-Service

Lists Windows services.

Start-Service -Name <serviceName>

Starts a Windows service.

Stop-Service -Name <serviceName>

Stops a Windows service.

Restart-Service -Name <serviceName>

Restarts a Windows service.

Get-EventLog -LogName <logname>

Views event logs.

Get-Alias

Lists all aliases.

Set-Alias <aliasName> <cmdlet>

Creates a new alias.

Remove-Item -Path alias:<aliasName>

Removes an alias.

Write-Host <text>

Prints text to the console.

Get-Help <cmdlet>

Shows help for a cmdlet.

Import-Module <moduleName>

Loads a PowerShell module.

Export-Csv -Path <file.csv>

Exports data to CSV file.

Import-Csv -Path <file.csv>

Imports data from CSV file.

Measure-Object

Measures properties of objects (count, sum, avg).

Select-Object

Selects properties of objects.

Where-Object

Filters objects based on condition.

Sort-Object

Sorts objects by property.

Out-File -FilePath <file.txt>

Sends output to a file.

Start-Job -ScriptBlock { }

Runs a background job.

Receive-Job -Id <jobId>

Retrieves job results.

Wait-Job -Id <jobId>

Waits for a job to complete.

Get-Job

Lists background jobs.

Stop-Job -Id <jobId>

Stops a background job.

Get-History

Shows command history.

Invoke-History -Id <historyId>

Runs a command from history.

New-ItemProperty -Path <regPath> -Name <propName> -Value <value>

Creates or sets registry property.

Get-ItemProperty -Path <regPath>

Gets registry property.

Remove-ItemProperty -Path <regPath> -Name <propName>

Removes registry property.

Clear-Variable -Name <varName>

Clears a variable.

Get-Variable

Lists variables.

Set-Variable -Name <varName> -Value <value>

Sets variable value.

Read-Host -Prompt 'text'

Prompts user for input.

Test-Connection <hostname>

Sends ping requests to test network.

Get-NetIPAddress

Displays IP configuration.

New-Guid

Generates a new GUID.

Start-Process <program>

Starts a process/application.

Get-Location

Shows current directory.

Push-Location <path>

Saves current directory and changes to new path.

Pop-Location

Returns to previous directory saved by Push-Location.
